# 测试文件说明

本文件夹包含用于测试不同分块方式的示例文档。

## 文件列表

### 1. naive_test.txt
- **分块方式**: 朴素分块 (Naive)
- **内容**: 关于朴素分块方法的介绍文档
- **特点**: 简单的连续文本，适合测试固定大小分块
- **建议参数**: chunk_size=500, overlap=50

### 2. general_test.txt
- **分块方式**: 通用分块 (General)
- **内容**: 结构化的技术文档，包含多个部分
- **特点**: 有明确的段落结构，使用双换行符分隔
- **建议参数**: chunk_size=1000, overlap=100

### 3. book_test.txt
- **分块方式**: 书籍分块 (Book)
- **内容**: 人工智能教程，包含多个章节
- **特点**: 有明确的章节标题（第X章）
- **建议参数**: 默认参数即可

### 4. paper_test.txt
- **分块方式**: 论文分块 (Paper)
- **内容**: 学术论文格式的文档
- **特点**: 包含Abstract、Introduction、Methodology等标准论文结构
- **建议参数**: 默认参数即可

### 5. resume_test.txt
- **分块方式**: 简历分块 (Resume)
- **内容**: 个人简历
- **特点**: 包含个人信息、教育背景、工作经历等简历模块
- **建议参数**: 默认参数即可

### 6. table.csv
- **分块方式**: 表格分块 (Table)
- **内容**: 智能家居设备信息表
- **特点**: CSV格式，包含表头和30行数据
- **建议参数**: 默认参数即可

### 7. markdown_table_test.md
- **分块方式**: 表格分块 (Table)
- **内容**: 员工信息和项目统计表
- **特点**: Markdown格式的表格
- **建议参数**: 默认参数即可

### 8. qa_test_part1.txt & qa_test_part2.txt
- **分块方式**: 问答分块 (Q&A)
- **内容**: 智能家居常见问题解答
- **特点**: 标准的问答格式，每个问答对独立
- **建议参数**: 默认参数即可

## 使用说明

1. 创建知识库
2. 上传对应的测试文件
3. 为每个文件选择推荐的分块方式
4. 设置关键词提取数量（建议5个）和生成问题数量（建议3个）
5. 执行分块
6. 查看分块结果，验证分块效果

## 预期结果

- **朴素分块**: 应该生成固定大小的文本块，可能在句子中间切断
- **通用分块**: 应该按段落分块，保持段落完整性
- **书籍分块**: 应该按章节分块，每个chunk包含一个完整章节
- **论文分块**: 应该按论文结构分块（摘要、引言、方法等）
- **简历分块**: 应该按简历模块分块（教育背景、工作经历等）
- **表格分块**: 每行数据转换为键值对格式，包含完整的字段信息
- **问答分块**: 每个问答对作为一个独立的chunk
